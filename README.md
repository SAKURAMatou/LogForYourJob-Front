# LogForYourJob-Front

# 需求简述

为了方便在找工作时候记录投递的公司信息，方便对目标公司进行评价而做；工程使用**vue3**+**vite**构建，同时还使用**element-plus**,**pinia**,**vue-router**技术；

系统所有接口均为apifox的云端mock数据；登陆的账号密码随意输入
